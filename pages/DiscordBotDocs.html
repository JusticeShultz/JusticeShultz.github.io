    Overview

    Welcome to the world of Asteria, a land of mythic beasts and a mighty guild. Fight and quest for huge bounties, rewards and ranks! Make a name for yourself throughout the realms and fight to be the best!

    The Discord RPG Bot is written entirely in C# using the Discord.NET Core. Here you will find a full documentation of commands and getting the bot started on your server!

    It should be noted that the bot has some dependency roles and channels it will auto generate in your server on intialization, this means you should be careful when adding the bot to your server as it could be considered destructive.

    Overview

    Overview Function Prefixes Permission Level
    Prefix's The bot uses a - symbol to represent a command. You may also mention the bot as @Bot [Command]. (e.g. @Bot Help) - & @Bot
    Admin Vs. Bot Admin "Admin" refers to a server admin, someone who isn't part of the bots development. A "Bot Admin" refers to someone with internal access to the bot - like a game master. "Bot Admins" control the economy and handle issues relating to the bot.
    General Integrated Help This command will display the general help embed message in the chat channel the command was executed in. -help Anyone
    Initializing your server with the bot This command starts the servers initialization with the bot. This will generate several new Roles and Text Channels in the server. This command is somewhat obstructive so it is limited to admins only. Once this command is activated all of your users will get role updates as well if they have previous data. Not using this command will make the bot not function correctly in your server. -startserver Admin & Bot Admin
    Creating a profile Use this command to create your profile. The command should be laid out as -begin [Class] [Name] [Age], if your input is valid your profile should register correctly. -begin [Class] [Name] [Age] Anyone
    Checking your profile Use this command to check your profile. This will open your save file and write a embedded markup of your current stats to show you things like your needed XP and Level, etc. Additionally you may tag someone like @Robin and it will show their profile.
    -profile
    -profile [User]
    -p
    -p [User]    		 Anyone
    Seeing available classes Get a list of all available classes to be in the server. -class
    -classes    		 Anyone
    Spawning enemies This command may only be used in the level zones. Using this command will spawn a random enemy with random stats that you and other users may fight. -spawn
    -s
    -help spawn    		 Anyone
    Fighting enemies This command will be used to simulate 1 turn of a fight with a spawned enemy. Doing -fight [Count] will cause you to simulate fighting that many times unless you die or kill the monster in the process. Defeating a monster will provide currency and XP rewards. -fight
    -f
    -fight [Count]
    -f [Count]    		 Anyone
    Gold - Giving This command allows users to give other users their gold. Doing -gold give [User] [Amount] will give that user a certain amount of your gold to the specified user. If a bot admin uses this command they may put any number and it will not subtract from their gold amount. -gold give [User] [Amount]
    		 Bot Admin & Anyone
    Gold - Taking This is a bot admin only command. With this command the user may specify who to take gold from and then take a specified amount from them. The gold is not distributed to anyone and simply vanishes. -gold take [User] [Amount] Bot Admin
    Gold - Displaying This command simply displays your own gold. To view someone elses gold you must use -profile [User]. -gold
    -gold me    		 Anyone
    Displaying a servers user amount This command simply says how many users are in the server. -count
    -amount
    		 Anyone
    Display the servers leaderboard This command displays a leaderboard for every user in the server. This is a home server command only. -leaderboard
    -lb
    		 Anyone - must be in the development server
    Go on a quest This command allows you to go on a quest. You may quest every 5 minutes in the questing channel for small amounts of gold and guild gems. -quest
    -q
    		 Anyone
    Go on your daily quest This command allows you to go on a daily quest. You may quest every 24 hours in the daily blessings channel to earn large amounts of gold and guild gems. -daily
    -d
    		 Anyone
    Switch your class You may use this command to switch your class. Note, this command will cost 500 gold coins to use. This command is purely cosmetic and your base stats will not change. -switchclass
    -changeclass
    		 Anyone - 500 Gold Cost
    Open the shop Use this command in the guild shop to get a list of every buyable item the shop keeper has in stock. -shop
    -sh
    		 Anyone
    Buy an item from the shop Use this command in the guild shop to buy an item by it's ID. Open the shop to see the item ID's in the brackets([]). Note, using this command will buy the item if you have the funds, removing the funds from your account after the transaction is completed. -buy [ID] Anyone
    Display the event shop Use this command to see the special event items you may buy. Note, buying works the exact same as it does for normal items. Do -buy [ItemID] to purchase an event item. -event shop Anyone
    Display help for the current guild special event Use this command to get general help commands for the servers ongoing special event. -event help Anyone
    Display every active special event Use this command to display every ongoing special event in the server. This is admin only due to the spam in generates if used more than once. -event Admin
    Spam a user with pings Use this command if you don't like someone. -spam [User] [Amount] Admin
    Prune messages from a channel Use this command to remove a certain amount of messages from a channel. Please note, abuse of this command will result in a blacklisting of you and your servers use with our bot. This command causes slowdowns for everyone else if used in huge amounts. -prune [Amount] Admin
    Sending notices Use this command to send announcements through the bot. Please note your announcement must be kept somewhat short. If it is long you will need to send the message yourself. -notice [Text] Admin
    Add Damage Use this command to give the capability to 1 shot anything. -thanos Bot Admin
    Show your profile picture Use this command to display your profile picture. This is a simple tool command to hook to Discords API. -profilepicture
    -pp    		 Anyone
    Show a users profile picture Use this command to display a specified users profile picture. This is a simple tool command to hook to Discords API. -profilepicture [@User]
    -pp [@User]    		 Anyone
    Ping pong the bot Simple command that debugs connection of the system. -ping
    -ping1    		 Anyone
    Howdy! Say hello to the bot. Essentially just the -ping command. -hello Anyone
    Debug Embed Check to make sure embed messages format on your device. -embed Anyone
    Display your gems This command will simply display your guild gem count. -gems
    -gem
    -guildgems
    -guildgem    		 Anyone
    Make a "petition" This command will simply generate a message with the users text with some additional embed candy. -pe Anyone
    Force admin This command will force a Bot Admin into an Admin role on the server. Note, this is only used for extreme cases where the bot needs direct moderation. We will never use this command to be destructive. -Reee Bot Admin
    Force leave server This command will force the bot to leave the server. Admins please kick the bot if you'd like it gone, this command is for extreme cases like above! -leaveserver Bot Admin
    Display active users in server This command will display how many users are not marked "Afk" by Discord. -online Anyone
