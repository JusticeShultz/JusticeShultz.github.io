Overview

Welcome to the world of Asteria, a land of mythic beasts and a mighty guild. Fight and quest for huge bounties, rewards and ranks! Make a name for yourself throughout the realms and fight to be the best!

The Discord RPG Bot is written entirely in C# using the Discord.NET Core. Here you will find a full documentation of commands and getting the bot started on your server!

It should be noted that the bot has some dependency roles and channels it will auto generate in your server on intialization, this means you should be careful when adding the bot to your server as it could be considered destructive.

Overview

Command Function Prefixes Permission Level

Help

 This command will display the general help embed message in the chat channel the command was executed in. -help, -h

Anyone

Start Server

 This command starts the servers initialization with the bot. This will generate several new Roles and Text Channels in the server. This command is somewhat obstructive so it is limited to admins only. Once this command is activated all of your users will get role updates as well if they have previous data. Not using this command will make the bot not function correctly in your server. -StartServer, -BeginServer

Admin

Create a Profile

 Use this command to create your profile. The command should be laid out as -begin [Class] [Name] [Age], if your input is valid your profile should register correctly. -begin [Class] [Name] [Age]

Anyone

Check Your Profile

 Use this command to check your profile. This will open your save file and write a embedded markup of your current stats to show you things like your needed XP and Level, etc. Additionally you may tag someone like @Robin and it will show their profile. -profile, -profile [User], -p, -p [User]

Anyone
